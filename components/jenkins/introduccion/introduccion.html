Introducción a Jenkins

Es un servidor de automatización, que permite desplegar proyectos fácilmente.

Está programado en java y por lo tanto el sistema necesita java para ejecutarlo

Permite la instalación de plugins.

Instalación y bootstrap junto a docker y java 11:

  • Instalación de Java 11, variable de entorno home y actualización de PATH
  • Instalar Jenkins
  • Instalar Docker desktop(Opcional)
  • Instalar Chocolatey(Opcional)

NOTA: La nueva versión de jenkins utiliza las versiones de java 11 a la 17

Se puede instalar jenkins desde un documento con extensión .msi en Windows o ejecutar el documento con extensión .war con java mediante comandos.


#Comando para ejecutar jenkins
java -jar jenkins.war httpPort=8080
